# NoBOOM: A collection of datasets from chemical plants.

## Time Series Anomaly Detection ([tsad](https://github.com/wagner-d/noboom/tree/main/src/noboom/tsad))
Noboom contains datasets for evaluating anomaly detection methods on time series data. 
Version 1.0.0 of the data is available at [kaggle](https://www.kaggle.com/datasets/faebs94/noboom-anomaly-detection-in-chemical-processes). 
Details are available in the [paper](https://openreview.net/forum?id=qiLboR0ocm).
